# 🗺️ MosGuessr

### Докажи, что ты не понаехал

Ты можешь отличить Бибирево от Бирюлёво? Знаешь, где кончается Садовое и начинается ТТК? MosGuessr — это безжалостный тест на знание Москвы по панорамам. Здесь нет морских побережий или горных вершин, только хрущёвки, пробки и вечная стройка.

**Суть игры:** Тебя выкидывают в случайную точку столицы. Угадай по панораме, где именно ты находишься. Чем точнее ткнёшь в карту — тем больше очков. 5 раундов, чтобы доказать матери, что ты не зря родился в ЦАО (или выучить город по пути на работу).
