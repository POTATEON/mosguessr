# models/

Тут живут классы SQLAlchemy. Каждый класс = одна таблица в базе.

user.py     - User (логин, почта, пароль, аватарка)
location.py - Location (название точки Москвы, координаты, путь до панорамы)
game.py     - Game (кто играл, сколько очков) и Round (данные одного раунда)

Если добавил новый класс - не забудь сделать миграцию:
flask db migrate -m "описание"
flask db upgrade
