# app/

Здесь лежит весь код приложения.

models/   - описание таблиц базы данных (User, Game, Location)

routes/   - обработчики урлов, принимают запросы и отдают ответы

services/ - вспомогательные функции: посчитать расстояние, сохранить файл

static/   - картинки, css, js и папка для загруженных админом панорам

templates/- html шаблоны страниц

__init__.py - фабрика приложения, тут всё собирается воедино и запускается
